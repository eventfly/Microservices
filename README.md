# Microservices Monorepo

## Installation
1. Download and install Docker/Docket Desktop
2. `git clone github.com/eventfly/Microservices.git`
3. `cd Microservices`
4. `docker compose build`
5. After build, run using `docker compose up`

## Port Mapping
```
- 3000: Events Service
- 3001: Auth Service
- 3002: Org Service
```
